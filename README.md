# ES-rescue-and-recovery-plugin
ES plugin for new jobs involving search and rescue of passengers or outfits recovery in derelict ships.

This plugin will spawn jobs on the job board directing you to a system or a cluster to find a disabled ship, board it and either rescue its passangers or take a specific outfit from it.

Similar to bounty hunter jobs, the difficulty will increase based on number of previous jobs completed and/or combat rating.
